Noir is a library that contains various helper classes and solutions that allow game developers to add deep and complex features to their game without requiring a lot of work up front.

## Features

<div class="grid cards" markdown>

- :material-hexagon-multiple-outline: __[Automation]__ – Releases, API documentation and more handled for you
- :material-code-braces: __[Productivity]__ – Build your package in Unity, alongside your samples, in the same repository
- :material-package-variant-closed: __[OpenUPM]__ – Publish releases to [OpenUPM](https://openupm.com/) automatically
- :material-file-document-outline: __[Documentation]__ – Documentation is published to GitHub Pages and powered by [Material for mkdocs](https://squidfunk.github.io/mkdocs-material)

</div>
