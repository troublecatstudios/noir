# SortingLayerAttribute

<!-- tc:namespace Noir.Attributes -->

<!-- tc:assembly Noir.dll -->

Displays a dropdown of available sorting layers for the annotated field in the Unity Inspector.
This attribute is typically applied to a [int](https://learn.microsoft.com/en-us/dotnet/api/System.Int32?view=net-7.0) field to make it easier
            to select and assign a sorting layer without manually typing its name or ID.
